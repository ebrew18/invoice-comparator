# Invoice Comparator

Browser-based commission reconciliation tool for comparing the Commission Source Report against vendor invoices.

**Live app:** https://ebrew18.github.io/invoice-comparator

## Features
- Drag & drop Excel file upload
- Auto-detects vendor week and matching source report tab
- Compares Sales, Returns, and Net amounts per invoice
- Flags matches, mismatches, and missing invoices on either side
- Downloads a multi-sheet Excel report

## Usage
Just open `index.html` in any browser — no install required.

## Deploy
This is a static single-file app. GitHub Pages serves it automatically.
